# Russell Spangler — Writing & Editorial Portfolio

This repository collects writing, editing, layout, and document-design samples from
my work in infrastructure and environmental consulting (primarily with Gannett
Fleming).

## How this repo is organized

Files are grouped by **type of deliverable** rather than by project:

| Folder | Contents |
|---|---|
| [`Writing_Samples/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/Writing_Samples) | Curated excerpts with my own notes on what I specifically contributed (writing, editing, layout, graphics) |
| [`Work_Flow/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/Work_Flow) | A numbered walkthrough (draft → layout → client review → print) showing my document production process |
| [`Project_Sheets/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/Project_Sheets) | One-page project summary sheets |
| [`Documents/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/Documents) | Full source reports, studies, and EIS documents, including two project subfolders (AASHTO_TIG, Baudette-Rainy_River) |
| [`Presentations/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/Presentations) | Presentation-format materials |
| [`Interactive_PDF_and_Database_Interface_Design/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/Interactive_PDF_and_Database_Interface_Design) | Dam safety case studies and the interactive dam owner toolbox |
| [`Public_Outreach/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/Public_Outreach) | Public-facing outreach materials |
| [`Flowcharts/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/Flowcharts) | Process flowcharts |
| [`CD_Labels/`](https://github.com/rspangler84/rspangler84.github.io/tree/main/CD_Labels) | CD label design work |

## Naming convention

Folder and file names use underscores in place of spaces for cleaner,
more portable links.
